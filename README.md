# SIA-IDN
> A fast, secure, and responsive real-time Punycode & Internationalized Domain Name (IDN) Converter.

SIA-IDN is a 100% client-side static web application designed to instantly convert Unicode domain names (e.g.  শাহরিয়ারইবনেআলম.বাংলা or `café.com`) into their ASCII-compatible Punycode format (`xn--...`) and vice versa. Built using HTML5, JavaScript, and Tailwind CSS.

## ✨ Features
- **Real-time Auto-Conversion:** Converts instantly as you type or paste text without needing to click any button.
- **Robust Bengali Support:** Fully optimized with **Unicode Normalization (NFC)** to flawlessly handle complex characters, conjuncts, and vowel signs in Bengali scripts.
- **One-Click Copy:** Minimalist and non-intrusive copy buttons with visual feedback (`Copied! ✓`) without annoying alerts.
- **100% Client-Side & Secure:** No servers involved. Your data never leaves your browser, ensuring absolute privacy.
- **Fully Responsive Design:** Seamlessly adapts to mobile, tablet, and desktop screens thanks to Tailwind CSS.
